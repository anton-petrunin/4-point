There are two master files, `arXiv.tex` and `msb.tex`, for the arXiv and *Math Sbornik* versions, respectively.
The corresponding Russian versions are `arXiv-ru.tex` and `msb-ru.tex`.

To collect all the required files, run:

`tar -cvf arXiv.tar --files-from arXiv.txt`<br>
`tar -cvf msb.tar --files-from msb.txt`<br>
`tar -cvf arXiv-ru.tar --files-from arXiv-ru.txt`<br>
`tar -cvf msb-ru.tar --files-from msb-ru.txt`<br>
